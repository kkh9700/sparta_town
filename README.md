# sparta_town
# 프로젝트 소개
유니티를 이용해 게더를 모방하는 프로젝트입니다.

타일맵을 사용하여 배경을 만들고 각종 UI를 제작합니다.

# 개발 기간
2023.09.4 ~ 2023.09.07
